# Armis Centrix™ Inspection Categories

| Category | Description | Expected Checks |
|---|---|---|
| Collectors | Health and operation of Armis data collectors, ingestion pipelines, and API data retrieval processes | 20+ |
| Asset inventory | Verification of asset discovery completeness, ownership, and lifecycle tracking | 25+ |
| Device intelligence | Validation of device classification, manufacturer, model, OS, firmware, and risk scoring | 30+ |
| Alerts | Detection coverage, alert severity distribution, alert lifecycle, and response tracking | 40+ |
| Risk / vulnerabilities | Identification of CVEs, exploitable vulnerabilities, and remediation posture | 30+ |
| Policies | Policy configuration, enforcement coverage, policy usage, and policy conflicts | 25+ |
| Boundaries | Segmentation enforcement, cross-boundary communication monitoring, and boundary coverage | 20+ |
| Integrations | Health and data synchronization of external integrations (SIEM, CMDB, NAC, ITSM, etc.) | 20+ |
| Users / access | Platform user accounts, authentication status, and account lifecycle management | 15+ |
| Roles | Role definitions, role assignments, and privilege distribution across the platform | 10+ |
| Permissions | Fine-grained permission assignments and privilege escalation risk analysis | 15+ |
| API / platform health | API availability, authentication validity, rate limits, and ingestion reliability | 15+ |
| Networks | Network discovery coverage, subnet mapping, and network asset association | 20+ |
| Network behavior / communications | Device communication patterns, protocol usage, and anomalous network activity | 20+ |
| Discovery / sensors | Visibility architecture including discovery sources and monitoring coverage | 15+ |
| Sites / locations | Site hierarchy, geographic mapping, and asset-to-site assignment accuracy | 10+ |
| Tags / categories | Metadata tagging completeness and taxonomy consistency | 10+ |
| Groups | Logical asset grouping used for policy scope and operational segmentation | 10+ |
| Certificates | Certificate inventory, expiration tracking, and trust chain validation | 10+ |
| Firmware / OS exposure | Operating system and firmware lifecycle, unsupported versions, and patch posture | 15+ |
| Threat intelligence | Threat feed ingestion and enrichment mapping to assets and alerts | 10+ |
| Exposure surface | Internet-facing assets, exposed services, and external attack surface monitoring | 15+ |
| Data quality | Data integrity validation including null fields, duplicates, and parsing errors | 20+ |
| Governance | Ownership attribution, policy accountability, and operational maturity | 15+ |
| Compliance posture | Compliance monitoring against regulatory and internal security frameworks | 10+ |
| VIPR automation | Automated remediation rules, enforcement actions, and response automation effectiveness | 20+ |
| Workflows / remediation | Incident response workflow tracking and remediation lifecycle management | 15+ |
| Operational metrics / KPIs | Platform operational metrics measuring detection and response effectiveness | 10+ |
| Trend analysis | Day-over-day, week-over-week, month-over-month, quarter-over-quarter, and year-over-year trends | 10+ |
